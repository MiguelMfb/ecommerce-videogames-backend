# Ecommerce Backend for Videogames

## Prerequisites
- Node.js
- MongoDB

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example`.
4. Start MongoDB.

## Development
Run the server in development mode:
```bash
npm run dev
```

## Production
Build and start the server:
```bash
npm run build
npm start
```
